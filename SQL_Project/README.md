# SQL Project – Sales & Customer Analysis

This project demonstrates the use of SQL to analyze sales, product, and customer data to support data-driven business decisions.

## Objective
To analyze sales performance and customer behavior by identifying revenue drivers, profitability patterns, and customer contribution segments using SQL.

## Key Analysis Performed
- Calculated total revenue, profit, profit margin, and return rates at product and brand level using CTEs and window functions
- Identified top performing products using ranked queries
- Analyzed high returns versus low-sales products to identify risk areas affecting profitability
- Assessed the impact of returns on gross revenue versus net revenue 
- Segmented customers based on purchase value and frequency (High, Medium, Low)
- Identified repeat and high-value customers contributing significantly to revenue
- Used aggregations, joins, and conditional logic, CTEs and window functions for structured and efficient analysis

## Key Insights
- High-revenue products can still show low profit margins due to high returns, highlighting operational risk areas
- A small group of repeat customers contributes disproportionately to total revenue, emphasizing customer retention importance
- Customer segmentation helps differentiate high-value customers from low-impact buyers, supporting targeted sales strategies

## Tools Used
- SQL Joins, Aggregations, CASE statements, CTEs(Common Table Expressions) and Window Fuctions(RANK, DENSE_RANK, SUM OVER, etc.)
