# SQL Project – Sales & Customer Analysis

This project demonstrates the use of SQL to analyze sales, product, and customer data to support data-driven business decisions.

## Objective
To analyze sales performance and customer behavior by identifying revenue drivers, profitability patterns, and customer contribution segments using SQL.

## Key Analysis Performed
- Calculated total revenue, profit, profit margin, and return rates at product and brand levels
- Identified products with negative or low profitability
- Analyzed the impact of returns on gross versus net revenue
- Segmented customers based on purchase value and frequency (High, Medium, Low)
- Identified repeat and high-value customers contributing significantly to revenue
- Used aggregations, joins, and conditional logic for structured analysis

## Key Insights
- Certain products generate strong revenue but are impacted by returns and lower margins
- A smaller group of repeat customers contributes disproportionately to total revenue
- Customer segmentation helps differentiate high-value customers from low-impact buyers, supporting targeted sales strategies

## Tools Used
- SQL (Joins, Aggregations, CASE statements)
